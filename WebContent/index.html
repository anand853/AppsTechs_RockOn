Welcome to the Web ATM System.

Web ATM system performs deposit, withdraw and check balance functions from an online website. It maintains accounts/pins/balance/transactions only for each account holder.

ATM Functions:

Deposit: User enters account number, pin and amount. ATM validates them with existing accounts data. If match is found, system adds the balance to the account, error otherwise. ATM store this transaction record with date and type.

Withdrawal: User enters account number, pin and amount. ATM validates them with existing accounts data. If match is found, system deducts the balance from the account, error otherwise. ATM store this transaction record with date and type.

Check Balance: User enters account number and pin. System validates them with existing accounts data. If match is found, system displays the account balance, error otherwise. It also displays the last 5 transactions on this account number.
